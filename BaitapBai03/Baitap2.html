Bài tập 2: Kiểm tra số dương, âm hay bằng không

Yêu cầu người dùng nhập một số nguyên. Sử dụng câu lệnh if để kiểm tra xem số đó là số dương, số âm hay bằng không và in kết quả ra màn hình.

Copy đoạn mã mẫu sau và thay thế các dòng chú thích bằng các dòng mã tương ứng trên JDoodle.


#include<stdio.h>

int main() {
    int num;

    // Nhập số nguyên num
    
    // Kiểm tra:
    // Nếu num > 0, in ra màn hình: num là số dương.
    // Ngược lại, nếu num < 0, in ra màn hình: num là số âm.
    // Ngược lại, in ra màn hình: num bằng không.

}
