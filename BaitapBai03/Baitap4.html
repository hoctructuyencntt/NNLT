Bài tập 4: Đánh giá điểm học sinh

Yêu cầu người dùng nhập điểm của học sinh (A, B, C, D hoặc F). Sử dụng câu lệnh switch để đánh giá và in ra màn hình xếp loại tương ứng.

Copy đoạn mã mẫu sau và thay thế các dòng chú thích bằng các dòng mã tương ứng trên JDoodle.


#include<stdio.h>

int main() {
    char grade;
  
    // Nhập điểm grade (A, B, C, D hay F). Chú ý grade kiểu char

    // Nếu grade = 'A'
         // In ra: Xuất sắc!
    // Nếu grade = 'B'
         // In ra: Tốt!
    // Nếu grade = 'C'
         // In ra: Khá!
   // Nếu grade = 'D'
         // In ra: Trung bình!
   // Nếu grade = 'E'
         // In ra: Rớt!
   // Nếu grade = 'F'
         // In ra: Điểm không hợp lệ.
    }
    return 0;
}
